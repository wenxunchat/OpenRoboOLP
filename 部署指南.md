# OpenRobotVision 网站部署指南

## 🚀 免费云服务部署选项

### 方式一：Vercel 部署（最推荐）

1. 访问 https://vercel.com
2. 注册/登录账号（支持 GitHub、GitLab、Bitbucket）
3. 点击 "New Project"
4. 导入您的 GitHub 仓库
5. Vercel 会自动检测到这是一个 Vite 项目
6. 保持默认配置，点击 "Deploy"
7. 等待 1-2 分钟，部署完成！

### 方式二：Netlify 部署

1. 访问 https://netlify.com
2. 注册/登录账号
3. 点击 "Add new site" → "Deploy with Git"
4. 连接您的 GitHub 仓库
5. 配置构建命令：`npm run build`
6. 发布目录：`dist`
7. 点击 "Deploy site"

### 方式三：Cloudflare Pages

1. 访问 https://pages.cloudflare.com
2. 登录 Cloudflare 账号
3. 点击 "Create a project" → "Connect to Git"
4. 选择您的仓库
5. 构建命令：`npm run build`
6. 构建输出目录：`dist`
7. 点击 "Save and Deploy"

## 📦 项目配置说明

- **构建命令**: `npm run build`
- **输出目录**: `dist`
- **框架**: Vite + React
- **路由模式**: SPA（需要重写配置）

## 🔧 本地预览构建

```bash
npm run build
npm run preview
```

## 📝 配置文件说明

- `vercel.json` - Vercel 部署配置（SPA 路由重写）
- `netlify.toml` - Netlify 部署配置（SPA 路由重写）
