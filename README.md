# OpenRobotVision

一个专注于机器人视觉与自动化外设控制的开源平台，提供完整的硬件模块、软件工具和仿真环境。

## 🚀 功能特性

- **视觉模块**：高性能机器人视觉感知解决方案
- **外设控制**：自动化外设控制器支持
- **仿真调试**：完整的仿真环境和调试工具
- **跨平台**：支持多种操作系统和开发环境

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **图标库**：Lucide React Icons

## 📦 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 📁 项目结构

```
├── src/              # 源代码目录
│   ├── components/   # React 组件
│   ├── pages/        # 页面组件
│   ├── assets/       # 静态资源
│   └── App.tsx       # 主应用组件
├── public/           # 公共资源
├── index.html        # HTML 入口
├── vite.config.ts    # Vite 配置
├── tailwind.config.js # Tailwind 配置
└── tsconfig.json     # TypeScript 配置
```
