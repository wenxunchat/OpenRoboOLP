# Plugins

Third-party extensions and community plugins.

## Plugin API (Planned)
- Python-based: inherit from `orolp.PluginBase`
- C++ based: dynamic library with `create_plugin()` export

## Categories
- **Process**: Welding, painting, grinding path generators
- **Sensor**: Vision-guided calibration, force control
- **Communication**: PLC drivers, fieldbus adapters
- **UI**: Custom panels, themes, exporters
